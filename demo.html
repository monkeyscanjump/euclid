🏗️ Architecture

  • Framework Agnostic: Works with React, Angular, Vue, vanilla HTML
  • Multi-Chain: Address book pattern, multiple wallet support
  • Headless Controllers: Data logic separated from UI
  • Type Safe: Full TypeScript with API integration
  • Performance: Lazy loading, scoped CSS, minimal bundles
