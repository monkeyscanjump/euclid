üìö Architecture Overview

üèóÔ∏è Design Principles

  • Framework Agnostic: Works with React, Angular, Vue, and vanilla HTML
  • No Framework Lock-in: Standards-based Web Components
  • Multi-chain by Design: Address Book pattern, not single wallet connection
  • Headless Controllers: Separate data logic from UI components
  • Reactive State: Stencil Store for predictable state management
  • Type Safety: Full TypeScript integration with API types
